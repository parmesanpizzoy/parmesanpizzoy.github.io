# Headphones Store Website
A modern, responsive, and visually stunning e-commerce web application dedicated to premium audio gear and headphones. This project showcases clean UI/UX design, interactive product catalogs, and a seamless shopping cart experience.

## 🚀 Live Demo
Check out the live deployment of the store here:
👉 (https://parmesanpizzoy.github.io)

## ✨ Key Features
- **Premium UI/UX:** Sleek, dark/light-themed minimalist aesthetics tailored for luxury tech products.
- **Dynamic Product Catalog:** Filter headphones by categories (Wireless, Noise-Canceling, Audiophile, Gaming).
- **Interactive Shopping Cart:** Real-time price calculation, item increments, and persistent cart state.
- **Product Spotlight:** Immersive detailed pages for high-end flagship headphones featuring specifications and high-res imagery.
- **Responsive Layout:** Optimized flawlessly for mobile, tablet, and desktop viewports.
